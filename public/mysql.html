MySQL Interview Questions

1. What is MySQL?

MySQL is a multithreaded, multi-user SQL database management system which has more than 11 million installations. It is the world's second most popular and widely-used open source database. The term "My" in MySQL is coined after the name of the daughter of co-founder Michael Widenius, and "SQL" stands for Structured Query Language. MySQL is free of cost for developers to use, but enterprises have to pay a license fee to Oracle.

MySQL was initially owned by a for-profit firm MySQL AB, then Sun Microsystems bought it, and eventually, Oracle acquired Sun Microsystems, making Oracle the current owner of MySQL.

MySQL is an Oracle-supported Relational Database Management System (RDBMS) based on structured query language. It supports a wide range of operating systems, including Windows, Linux, and UNIX. While it is capable of developing various applications, it is commonly used for web applications and online publishing. MySQL is a fundamental part of the LAMP (Linux, Apache, MySQL, and PHP) stack, which is a platform used for web development.

4. What is the difference between MySQL and SQL?

SQL, which stands for Structured Query Language, is a standard query language used to interact with databases, including MySQL. MySQL, on the other hand, is a specific database management system that stores various types of data and ensures its safe handling.

SQL is a computer language used for querying, updating, and managing relational databases, while MySQL is a software or application that implements the SQL language to enable data handling, storing, deleting, and modifying data.

In summary, SQL is a language, whereas MySQL is an application that uses SQL as the query language to interact with the database.

5. What is the difference between the database and the table?

There is a major difference between a database and a table. The differences are as follows:

  • Tables are used to represent the division of data within a database. They are used to group related data in rows and columns.
  • A database, on the other hand, is a collection of tables and other objects, such as views, indexes, and procedures, that are organized and structured to store and manage data.
  • Tables are part of the database and are used to create datasets within the database. However, the entire database includes multiple tables, relationships, and data stored in various formats and structures.

In summary, tables are a way to organize and group data within a database, while the database itself is a collection of tables and other objects used to store and manage data efficiently.

9) How to check the MySQL version?

We can check the MySQL version on Linux using the below command:

mysql -v

If we use MySQL in windows, opening the MySQL command-line tool displayed the version information without using any flags. If we want to know more about the server information, use the below statement:

SHOW VARIABLES LIKE "%version%";

It will return the output with additional version information about the installed MySQL software like innodb_version, protocol_version, version_ssl_library, etc.

10) How to add columns in MySQL?

A column is a series of cells in a table that stores one value for each row in a table. We can add columns in an existing table using the ALTER TABLE statement as follows:

ALTER TABLE table_name
ADD COLUMN column_name column_definition [FIRST|AFTER existing_column];

16. How to create a table in MySQL Workbench?

To create a table in MySQL Workbench, follow these steps:

  1. Launch MySQL Workbench and go to the Navigation tab.
  2. Click on the Schema menu to view all the previously created databases.
  3. Select a database and double-click on it to show sub-menus.
  4. Choose the Tables option from the sub-menu.
  5. Right-click on Tables and select Create Table option, or use the create new table icon.
  6. A new popup screen will appear, where you need to fill in the table name and column details.
  7. Click on the Apply button and then the Finish button to complete the table creation.

17. How to change the table name in MySQL?

To rename a table in MySQL, you can use the RENAME TABLE statement with the following syntax:

RENAME TABLE old_table TO new_table;

If you want to change more than one table name, you can use the following syntax:

RENAME TABLE old_tab1 TO new_tab1, old_tab2 TO new_tab2, old_tab3 TO new_tab3;

18. How to change the database name in MySQL?

To rename a database in MySQL, follow these steps:

  1. Create a new database in MySQL.
  2. Use the mysqldump shell command to create a dumped copy of the selected database.
  3. Import all the data into the newly created database using the mysql command.
